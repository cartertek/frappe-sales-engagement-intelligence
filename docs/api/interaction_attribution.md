# Interaction Attribution API

Methods:

- `create_interaction_attribution(payload)`
- `get_interaction_attributions(prospect=None, crm_lead=None, crm_deal=None)`

Attribution records can link thesis, offer, asset, source arena, channel, response category, SEI Prospect, CRM Lead, and CRM Deal where fields exist. These endpoints do not send outreach and do not replace Frappe CRM activity records.
